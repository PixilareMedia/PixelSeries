Pixel Craft Season #6

Info

Season #6 of Pixel Craft this season we are starting in The Wild Update and playing in the same world for as long as we feel like keeping this world. We've also got a bunch of custom stuff that we are adding to the server to enhance our gameplay experience.

We have an optional things for people to use if the want and one required thing. The optional things is a Fabric client. The required thing is a Custom Resource Pack that we use for the custom things that we wanted to add to the server. You can get all of these from Resource Packs Here and Clients Here.
