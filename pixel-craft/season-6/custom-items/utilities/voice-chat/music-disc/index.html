Pixel Craft Season #6

Music Discs

This is what is required for the Music Discs recipe.

To make a custom music disc you need to run a few commands.
1. Follow the 1st screen shot but replace the blue text with your URL."
2. Click on the green text that says "[Put on music disc]".
3. Fill out the yellow text int the 3rd screen shot with the name of the disc and optional the green text as well for a different range.

During this you should be holding a music disc so you can apply it when prompted to.
Alternatly when upload you can ask the server admin if they can upload it to the server directly and send you the ID for the blue text in the 3rd screen shot
