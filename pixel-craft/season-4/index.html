Pixel Craft Season #4

Info

Season #4 of Pixel Craft this season we have gone back to the roots a bit better that before. As now we only have things that help improve the vanilla gameplay so one play sleep, and a few others from vanilla tweaks. Plus a few plugins to link the game chat to our discord and handle the player shops.

We have three optional things for people to use if the want and one required thing. Two of the optional things are a Forge & Fabric along with a Vanilla Tweaks Resource Pack. The required thing is a Custom Items Resource Pack that we use for the Diamond shards and other custom things that we wanted to add to the server. That you can get from Resource Packs Here and Clients Here.
